# Superstore Sales Dashboard

## Project Overview
This Power BI dashboard analyzes Superstore sales data to uncover insights related to sales, profit, customers, and regional performance.

## Tools Used
- Power BI
- Excel / CSV
- Data Cleaning
- Data Visualization

## Dashboard Features
- Total Sales Analysis
- Profit Analysis
- Regional Performance
- Category-wise Sales
- Customer Insights
- Interactive Filters

## Key Insights
- Identified top-performing regions
- Analyzed profit trends
- Compared category performance
- Tracked sales growth
